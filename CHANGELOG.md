## 5. 留影簿 Guestbook（重磅升级）

### 5.1 数据库
- 使用 **Firebase Firestore** 存储留言和回复
- 数据云端保存，不依赖浏览器缓存

### 5.2 注册 / 登录
- 昵称 + 六位数字密码
- 点击左上角 **zᶻ** → 登录/注册弹窗（18% 灰背景）
- 忘记密码？→ 输入昵称直接重置
- 登录后称呼框自动预填
- 登录后自动展示自己的私密留言

### 5.3 留言功能
- 称呼 + 留言内容 + 相纸颜色（暖白/焦糖/蓝晒）
- **隐私选项**：默认「☑ 他人可见」，取消勾选仅 ROYZHAO 可见
- **天气定位**：自动获取留言者城市、天气、温度
- **卡片动画**：hover 抬起 + 阴影加深
- **卡片布局**：右上角日期 + 左下角时间 + 右下角天气位置
- **分页**：每页 6 条，翻页按钮

### 5.4 回复功能
- 点击留言卡片 → 弹出 18% 灰模态框 → 查看详情 + 回复
- 回复隐私：默认「☑ 所有人可见」，取消勾选仅彼此 + ROYZHAO 可见

### 5.5 站长功能
- 点击留言区 **ROYZHAO** → 输入密码 → 管理模式
- 密码：`Zrh20060727`
- 管理模式：查看所有留言（含私密）、**置顶**消息
- **置顶**：点击留言卡片 → 底部「📌 置顶」按钮

### 5.6 查找留言 🔍
- 输入昵称 → 显示该用户的所有公开留言
- 登录后搜自己 → 公开 + 私密都显示

### 5.7 删除留言
- 登录用户可删除自己发布的留言（含所有回复）

## 7. 管理后台 admin.html

| 标签 | 功能 |
|------|------|
| Hero 首页 | 编辑标签、标题、描述、图片路径、标注 |
| 底片册 | 拖拽排序、编辑字段、添加/删除/复制照片、批量编辑 |
| 关于我 | 编辑自我介绍、设备、底片、地区、联系方式、社交链接 |
| 留影簿 | 编辑副标题和标题 |
| 留言管理 | 查看/删除所有留言，刷新、清空 |

- 快捷键：`Ctrl+1~5` 切换标签，`Ctrl+S` 保存，`Ctrl+E` 导出 JSON
